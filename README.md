## 在分发过程中保护 Ruby 代码版权

如同所有解释型语言一样，Ruby 项目的源码在分发过程中很容易被阅读。如果产品需要提供自托管部署，便很难保护项目的版权。本 Talk 将总结现有的保护源码的方式，并尝试引入一些新的技巧来保护源代码的版权。
